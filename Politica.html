Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo Estante Virtual lida com as informações dos usuários.

Coleta de Dados

O aplicativo Estante Virtual não coleta, armazena ou compartilha quaisquer dados pessoais dos usuários. Todas as informações inseridas no aplicativo são salvas localmente no dispositivo do usuário.

Uso de Informações

Como o aplicativo não coleta dados pessoais, não há uso, compartilhamento ou envio de informações a terceiros.

Anúncios

O aplicativo não exibe anúncios de terceiros.

Login e Conta

Não é necessário criar uma conta ou realizar login para utilizar o aplicativo.

Alterações nesta Política

Esta política pode ser atualizada futuramente. Recomenda-se que o usuário revise periodicamente esta página para se manter informado sobre quaisquer alterações.
